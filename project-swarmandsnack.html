Swarm and Snack

ASP.NET Core 9 SignalR Real-time Multiplayer

A head-to-head arena game where each player commands a swarm of underlings in a deterministic .NET 9 simulation. SignalR keeps 50 ms ticks in sync while client-side prediction and Canvas 2D ensure the action feels responsive in the browser.

Explore the build

Key Features

Deterministic Simulation

Game logic is fully deterministic, allowing for perfect state sync between server and clients, and enabling features like instant replays and fair competitive play.

SignalR Real-Time Sync

SignalR is used to synchronize game state at 50ms intervals, ensuring all players see the same world and actions are reflected instantly.

Client-Side Prediction

To minimize perceived latency, the client predicts movement and actions, rolling back and correcting only if the server disagrees.

Canvas 2D Rendering

All visuals are rendered using the HTML5 Canvas 2D API for smooth, performant graphics in the browser.
