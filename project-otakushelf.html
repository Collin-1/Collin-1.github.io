Otaku Shelf

React 18 Vite GraphQL Context API

A modern, responsive anime discovery platform inspired by Netflix's browsing experience. Built with React and powered by the AniList GraphQL API, it offers a seamless way to discover, search, and curate anime lists with a stunning Cyberpunk / Neo-Tokyo aesthetic.

View Live Demo

Key Features

Immersive UI Design

Features a dynamic Hero Banner and horizontal scrolling carousels showcasing Trending, Popular, and Upcoming anime. The Cyberpunk / Neo-Tokyo aesthetic with deep midnight blues and neon cyan accents creates an engaging browsing experience.

Smart Search

Real-time search functionality accessible directly from the navbar with URL synchronization for easy sharing. Find any anime instantly with responsive autocomplete suggestions.

Persistent Watchlist

Build and manage a personal watchlist that persists across sessions using LocalStorage. The "My List" feature ensures your curated collection is always available, even offline.

Rich Detail Modals

Clicking any anime opens a detailed modal with trailers, synopsis, and comprehensive metadata without navigating away from the browsing context. Seamless user experience maintained throughout.

Theme System

Built with a robust CSS variable architecture that allows for instant theme switching. Currently features Cyberpunk styling, with the foundation ready for additional themes like Light Mode, Sakura, or Manga Ink.

Fully Responsive

Mobile-optimized layout ensures a smooth experience across all devices. From desktop browsing to mobile scrolling, the interface adapts seamlessly to any screen size.

Technical Implementation

Custom CSS Architecture

Built without heavy UI libraries, using pure CSS3 with a comprehensive variable system for granular design control. Dynamic theme variables enable easy customization and future theme additions. 

:root {
  --color-bg-primary: #050510;
  --color-accent: #00f3ff;
  --navbar-height: 60px;
}

GraphQL Integration

Efficient data fetching from AniList's GraphQL API using modern fetch patterns. Demonstrates proper error handling, loading states, and optimized queries for external API consumption.

React Best Practices

Implements custom hooks for data fetching, Context API for global state management, and component composition for modular architecture. Clean separation of concerns with Cards, Modals, and Carousels.

Tech Stack

Frontend React 18, React Router v6
Build Tool Vite
Styling CSS3, CSS Variables
Data AniList GraphQL API
State Context API, LocalStorage
Deployment Netlify
