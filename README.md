# Collin-1.github.io

Personal portfolio website for Collin Sefako Makwala, focused on backend and full-stack engineering work.

Live site: https://collin-1.github.io/

## Overview

This repository contains a static portfolio site built with HTML, CSS, and small JavaScript utilities.

The site includes:

- A reusable sidebar navigation pattern across pages
- A project showcase with dedicated project detail pages
- A news hub with long-form article pages
- Contact and profile sections for hiring and collaboration
