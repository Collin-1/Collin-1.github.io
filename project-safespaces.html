FNB App of the Year 2025 Finalist

Safe Spaces

React Firebase PWA Location Services

A community-driven safety Progressive Web App (PWA) that connects individuals, local businesses, and law enforcement to prevent and respond to GBV (Gender-Based Violence) and public safety threats. The first platform in South Africa to merge verified safe zones, real-time alerts, and a proprietary crime-tracking API.

View Live App View Repository

Mission Statement

A platform meant to stay for good, for the good of everyone deserving and in need. Safe Spaces empowers communities to take an active role in public safety through technology, collaboration, and real-time data.

Key Features

Verified Safe Zones

Network of trusted locations where individuals can seek help during emergencies.

Panic Button & Voice SOS

Instant emergency alerts linked to police. Voice-activated SOS triggered by secret passphrase.

Crime-Tracking API

Real-time crime data for smart routing away from high-risk areas.

Real-Time Alerts

Location-based safety notifications and community-driven reporting system.

Cloud Evidence System

Automated evidence recording and secure cloud backup triggered by panic button.

Multilingual Support

Accessible in multiple South African languages for diverse communities.

What Makes Safe Spaces Unique

  • First-to-Market: First platform in South Africa to merge verified safe zones, real-time alerts, and a crime-tracking API in one comprehensive solution
  • Law Enforcement Integration: Strategic collaborations with police services for direct emergency response coordination
  • Community-Driven: Empowers businesses and citizens to actively participate in community safety
  • Progressive Web App: No app store required - accessible directly through browsers on any device
  • Evidence Preservation: Automated cloud backup ensures critical evidence is never lost
  • Inclusive Design: Multilingual support and accessibility features ensure everyone can benefit

Tech Stack

Frontend React, Progressive Web App
Backend Firebase (Auth, Firestore, Storage)
Location Services Geolocation API, Google Maps
Real-time Data Firebase Realtime Database
Crime Tracking Custom Crime-Tracking API
Voice Recognition Web Speech API

Social Impact

GBV Prevention

Directly addresses South Africa's gender-based violence crisis by providing immediate access to help and verified safe locations for those in danger.

Community Empowerment

Transforms passive community members into active safety participants, creating a collaborative approach to crime prevention and response.

Data-Driven Policing

Provides law enforcement with real-time data for better resource allocation and faster response times to emerging threats.
