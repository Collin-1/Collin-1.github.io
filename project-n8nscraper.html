n8n Automated Scraper

Python 3.10+ Playwright n8n Docker

Automated HTML scraper designed to run inside an n8n workflow or as a standalone CLI utility. Built with Playwright and Chromium, featuring manual anti-bot evasion and pagination support for capturing content from multi-page flows.

View Live Workflow View Repository

Key Features

Headless Chromium Scraping

Utilizes Playwright's async API with Chromium for reliable, headless browser automation. Handles dynamic content and JavaScript-rendered pages with ease.

Intelligent Pagination

Optional pagination support by clicking Next buttons via CSS selectors. Configurable max page limits prevent runaway scraping on large sites.

Anti-Bot Evasion

Implements manual anti-bot flags and playwright-stealth to bypass common detection mechanisms, ensuring successful data capture.

JSON Output

Emits structured JSON on stdout for seamless integration with n8n workflows or other automation tooling. Clean success/error handling included.

Docker Ready

Includes a Dockerfile that bundles n8n, Python, Playwright, and all system dependencies for easy deployment and consistent runtime environments.

CLI & Workflow Integration

Works as a standalone CLI utility or integrates directly into n8n Execute Command nodes for workflow automation scenarios.

CLI Usage

Run the scraper from the command line with flexible options:

python scraper.py <url> [--selector ".next-button"] [--max 5]
  • url: Starting page to load (required)
  • --selector: CSS selector for the Next button or link (optional)
  • --max: Maximum number of pages to capture (default: 1)

Example Output

Success payload:

{"status": "success", "pages": ["<html>...</html>", "<html>...</html>"]}

Error payload:

{"status": "error", "message": "Timeout 60000ms exceeded."}

Local Setup

python -m venv venv
venv\Scripts\activate          # Windows
pip install playwright playwright-stealth
playwright install chromium

Docker Deployment

Build and run the containerized version:

docker build -t n8n-scraper .
docker run -p 5678:5678 n8n-scraper

The Dockerfile extends node:20-bullseye, installs n8n globally, sets up Playwright with OS-level dependencies, and copies scraper.py into the container under a non-root user.

Troubleshooting Tips

Chromium Installation

Ensure Chromium binaries are installed via playwright install chromium before running the scraper locally.

Pagination Limits

Increase --max gradually. Some sites detect rapid pagination and may block requests.

Selector Precision

Use --selector values that uniquely match the Next button. Ambiguous selectors can cause click failures.
